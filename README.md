# DevOps Internship - Task 6: Host a Static Website with GitHub Pages

## 📌 Project Overview
This repository contains the deliverables for **Task 6** of the DevOps Internship. The objective of this task is to design a simple static HTML/CSS website and deploy it to the web for free using **GitHub Pages**. 

* **Live Website Link:** [https://hrishikeshg64.github.io/DevOps-Intern-Task-6/]
* **GitHub Repository:** [https://github.com/hrishikeshg64/DevOps-Intern-Task-6]
---

## 🛠️ Tools & Technologies Used
* **HTML5:** For core web page structure.
* **CSS3:** For styling and visual layout customization.
* **Git & GitHub:** For version control and source code management.
* **GitHub Pages:** For automated static web hosting.

---

## 🚀 Step-by-Step Implementation

1. **Created Project Files:** Developed a clean `index.html` file to act as the landing page and a `style.css` file to customize the look and feel with a modern layout.
2. **Initialized Git Repository:** Opened the project folder locally and ran `git init` to begin tracking files.
3. **Pushed to GitHub:** Created a new public repository on GitHub, linked it to the local environment, and pushed the files to the `main` branch.
4. **Configured GitHub Pages:** Navigated to the repository's **Settings > Pages** tab, selected the `main` branch as the deployment source from the root folder, and clicked save.
5. **Deployment Verification:** Confirmed successful deployment via the live hosting link provided by GitHub Actions.
